Leistungsbeschreibung

Disco 101 ist eine Sprachlern-App mit 404 englischen Phrasen in vier Themenbereichen (Basics, Urlaub, Job, Expat). Die App bietet Lernkarten, Tests und eine Wiederholungsfunktion.

Kapitel 1 jeder Unit ist kostenlos zugänglich. Weitere Kapitel können über In-App-Käufe freigeschaltet werden.
